# 凡尔登：沃堡守备战

中文 HTML5 试玩版，背景设定于 1916 年凡尔登战役中的沃堡。

## 试玩内容

- 第一、第三人称切换
- WebGL 三维渲染：220 米模块化甬道、雾气、动态灯光、积水、木箱与沙袋
- 敌军制服、钢盔与步枪细节参考了 NARA 收藏的 [1917 年德军步兵公开领域照片](https://commons.wikimedia.org/wiki/File:GermanInfantry1914.jpg)。
- 鼠标与 iPad 触控操作
- 战术定位盘、来弹方向提示与敌军弹道
- 三段任务：夺回弹药库、肃清西侧甬道、守住最后铁门

## 操作

- 移动：`WASD` / 方向键 / 左侧触摸摇杆
- 射击：鼠标左键 / 右下角“射击”
- 装填：`R` / “装填”按钮
- 视角：`V` / 右上角视角按钮

用浏览器直接打开 `index.html` 即可试玩。
